Membuat Fixed Asset

A. INPUT

(Tidak ada instruksi khusus)

B. LANGKAH KERJA

  1. Buka menu Accounting -> Assets -> Assets. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Klik tombol Create pada bagian atas-kiri form.

  1. Ubah Code of Asset dengan penomeran yang dikehendaki. Biarkan berisi / apabila menghendaki penomeran otomatis. Wajib diisi.
  2. Isi Name of Asset. Wajib diisi.
  3. Pilih Type. Wajib diisi.
  4. Beralih ke tab General.
  5. Isi Purchase Value. Wajib diisi.
  6. Pilih Start Date. Wajib diisi.
  7. Isi Salvage Value. Tidak wajib diisi.
  8. Pilih Asset Category. Wajib diisi.
  9. Pilih Analytic Account. Tidak wajib diisi.
  10. Pilih Partner. Tidak wajib diisi.
  11. Pilih Time Method. Wajib diisi.
  12. Isi Number of Years. Wajib diisi.
  13. Pilih Period Length. Wajib diisi.
  14. Pilih Computation Method. Wajib diisi.
  15. Aktifkan/ Deaktifkan Prorata Temporis. Tidak wajib diisi.
  16. Aktifkan/ Deaktifkan Prorate by Month. Tidak wajib diisi.
  17. Isi Date Min. to Prorate. Tidak wajib diisi.
  18. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data fixed asset akan terbuat dengan status Draft

Memodifikasi Fixed Asset

A. INPUT

  • Data fixed asset yang dapat dimodifikasi harus memiliki status Draft.
  • User yang akan memodifikasi harus memiliki akses untuk memodifikasi fixed asset.

B. LANGKAH KERJA

  1. Buka menu Accounting -> Assets -> Assets. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data fixed asset yang akan dimodifikasi. Abaikan jika data sudah dibuka.
  3. Klik tombol Edit pada bagian atas-kiri form.

  1. Ubah Code of Asset jika dibutuhkan. Wajib diisi.
  2. Ubah Name of Asset jika dibutuhkan. Wajib diisi.
  3. Isi dan sesuaikan Name of Asset jika dibutuhkan. Wajib diisi.
  4. Pilih dan sesuaikan Parent Asset jika dibutuhkan. Wajib diisi.
  5. Pilih dan sesuaikan Type jika dibutuhkan. Wajib diisi.
  6. Beralih ke tab General.
  7. Isi dan sesuaikan Purchase Value jika dibutuhkan. Wajib diisi.
  8. Pilih dan sesuaikan Start Date jika dibutuhkan. Wajib diisi.
  9. Isi dan sesuaikan Salvage Value jika dibutuhkan. Tidak wajib diisi.
  10. Pilih dan sesuaikan Asset Category jika dibutuhkan. Wajib diisi.
  11. Pilih dan sesuaikan Analytic Account jika dibutuhkan. Tidak wajib diisi.
  12. Pilih dan sesuaikan Partner jika dibutuhkan. Tidak wajib diisi.
  13. Pilih dan sesuaikan Time Method jika dibutuhkan. Wajib diisi.
  14. Isi dan sesuaikan Number of Years jika dibutuhkan. Wajib diisi.
  15. Pilih dan sesuaikan Period Length jika dibutuhkan. Wajib diisi.
  16. Pilih dan sesuaikan Computation Method jika dibutuhkan. Wajib diisi.
  17. Aktifkan/ Deaktifkan Prorata Temporis jika dibutuhkan. Tidak wajib diisi.
  18. Aktifkan/ Deaktifkan Prorate by Month jika dibutuhkan. Tidak wajib diisi.
  19. Isi dan sesuaikan Date Min. to Prorate jika dibutuhkan. Tidak wajib diisi.
  20. Klik tombol Save pada bagian atas-kiri form.

C. OUTPUT

  • Data fixed asset akan berubah sesuai dengan perubahan yang dilakukan.

Membuat Tabel Depresiasi

A. INPUT

  • Data fixed asset yang akan dibuat tabel depresiasi harus memiliki status Draft.

  • User yang akan membuat tabel depresiasi harus memiliki akses untuk membuat tabel depresiasi fixed asset.

B. LANGKAH KERJA

  1. Buka menu Accounting -> Assets -> Assets. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data fixed asset yang akan dibuat tabel depresiasi. Abaikan jika data sudah dibuka.
  3. Beralih ke tab Depreciation Board.
  4. Klik tombol Compute pada bagian kiri-atas tabel depresiasi.

C. OUTPUT

  • Tabel depresiasi fixed asset telah terbuat.

Membatalkan Fixed Asset

A. INPUT

  • Data fixed asset yang akan dibatalkan harus memiliki status selain Cancelled.

  • User yang akan membatalkan harus memiliki akses untuk membatalkan fixed asset.

B. LANGKAH KERJA

  1. Buka menu Accounting -> Assets -> Assets. Abaikan jika sudah berada pada menu yang dimaksud.
  2. Buka data fixed asset yang akan dibatalkan. Abaikan jika data sudah dibuka.
  3. Klik tombol Cancel pada bagian atas-kiri form.

  1. Klik tombol Ok pada pop-up konfirmasi pembatalan yang muncul.

  1. Pilih Reason pada pop-up Cancellation Reason yang muncul.

  1. Klik tombol Confirm pada pop-up Cancellation Reason.

  1. Klik tombol Ok pada pop-up konfirmasi pembatalan yang muncul.

C. OUTPUT

  • Status dari fixed asset akan berubah menjadi Cancelled
